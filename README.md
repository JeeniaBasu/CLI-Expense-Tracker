# CLI Expense Tracker

#### Video Demo: https://youtu.be/Eln-_1hOudI

## Overview
CLI Expense Tracker is a command-line application in C that allows users to record, store, retrieve, and analyze personal expenses. The project demonstrates strong foundational programming skills including modular design, file handling, and structured data management.

## Motivation
The goal of this project was to create a practical, real-world utility while applying key programming concepts from CS50x. Expense management is a common problem, and implementing it in C reinforced skills in input handling, file I/O, and program structure.

## Features
- Add, view, and delete expense records
- Search expenses by category
- Calculate total expenses
- Persistent storage using text files
